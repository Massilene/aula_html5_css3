PRINCIPAIS TAGS DO HTML5

TAG BÁSICA

< !DOCTYPE html > _ tipo de arquivo html
< html lang = "pt-br" > _ linguagem Português Brasil
< head > _ cabeçalho
< meta charset="utf-8" > _ permite acentuação
< title > _ título da página
< body > _ corpo do site
< p > _ define o parágrafo
< br > _ quebra linha
< h1 > _ o número varia de 1 a 6, define título de tamanhos diferentes; quanto maior o número, menor a letra.
< footer > _ rodapé

TAG DE NAVEGAÇÃO, SESSÃO E IMAGEM

< nav > _ navegação
< sections > _ separa seção de conteúdos (ex: notícia, esporte, entretenimento)
< img src "nome da imagem" > _ src mostra a origem da imagem
obs: site de imagens gratuitas: pexels.com

TAG DE DIVISÃO E LINK

< div > _ cria divisão (ex: cada imagem fica em uma div
< a href = "nome do link" > "descrição do link" < /a> _ a representa uma âncora (hiperlink). href é a referência.
obs:
- o link pode ser interno (dentro do próprio site) ou externo (fora do site ex: e-mail, telefone,outro site).
- # indica um id

OUTRAS TAGS

< ul > _ lista de itens com marcadores
< li > _ lista ordenada
